Hexo 安裝指南

2016-06-07

值得記念的第一篇，就記錄一下安裝 Hexo 的流程吧。

本機環境需求

  • Node.js
  • Git

安裝 Hexo

建立 Hexo 目錄

這樣 Hexo 就會在 這個資料夾建立好所有需要的檔案了。

啟動 Hexo 伺服器

試著在本機啟動 Hexo 吧：

$ hexo server

好了後在網頁輸入 http://localhost:4000，正常的話就會看到你的網站了！

建立 Github Repository

開一個新的 Github Repository，把 Blog 給 Github 管：

  • Repository name 寫 your_account.github.com

發佈到 Github 上

  • 修改 _config.yml，加上
  • 發佈：$ hexo deploy
    • 出現 Permission Deny：加 SSH key
  • 到 http://your_account.github.io/ 看 Blog 有沒有正常運行。
  • 如果想發佈到其他 Server 可以參考官方文件: Deployment
