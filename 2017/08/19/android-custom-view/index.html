Android 自定義 View/ViewGroup

2017-08-19
#Android #View

總之先從定義開始吧

View 是一個什麼概念？

  • Screen 上的一矩形區域，以一個 Rect 來表示
  • left, top 表示 View 相對於它的 parent View 的起點
  • width, height 表示 View 自己的寛高

View 繪製過程

  • Measure
    • protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec) {}
    • override onMeasure 來重新定義 View 的寛高
  • Layout
    • protected void onLayout(boolean changed, int left, int top, int right, int bottom) {}
    • 沒有子 View 需要排列，不需要做額外工作
    • ViewGroup 才會用到
  • Draw
    • protected void onDraw(Canvas canvas) {}
    • 啊就…畫 View 呀
  • 其他 methods:
    • requestLayout: View 重新調用一次 onLayout()
    • invalidate: View 重新調用一次 onDraw()
    • forceLayout: 標記 View 在下一次重繪，重新調用 layout 過程

ViewGroup?

  • Base class of layouts
  • Hold other View or ViewGroup

ViewGroup 繪製

  • Measure
    • protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec) {}
    • match_parent 或 xxxdp: setMeasuredDimension()
    • wrap_content: 遍歷所有的子 View，對每個子 View 進行測量，再算出最終 ViewGroup 的大小
  • Layout
    • protected void onLayout(boolean changed, int left, int top, int right, int bottom) {}
    • 對子 View 做排列
  • Draw
    • protected void onDraw(Canvas canvas) {}
    • 通常只需要調用 default method，讓子 View 去做事即可。

LayoutParams

  • 存儲子 View 在加入 ViewGroup 時的一些訊息
